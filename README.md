# software-downloader-for-xp
Executable window that downloads multiple XP software at once. (SEE THE "IMPORTANT" SECTION)


# Compatibility
The program is compatible with Windows XP SP2 and above. Executable compiled using Pyinstaller on Python 3.4.10 (you could run this on Windows 2000 Extended Kernel as well if you want). 

# IMPORTANT
As of now, I plan to stop working on the GUI and make the program rely on a command-line interface, which'll be published as an official release.
